# Villa El Alba

Sitio estatico listo para publicarse en Netlify.

## Nombre sugerido en Netlify

`villa-el-alba-mapa`

## Publicacion desde GitHub

1. Inicia sesion en Netlify.
2. Elige `Add new project` -> `Import an existing project`.
3. Conecta GitHub y selecciona este repositorio.
4. Usa estos valores:

   - Build command: dejar vacio
   - Publish directory: `.`
   - Site name: `villa-el-alba-mapa`

La URL esperada quedaria como:

`https://villa-el-alba-mapa.netlify.app`
